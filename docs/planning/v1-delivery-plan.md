# V1 Delivery Plan

이 문서는 V1 구현 작업을 issue/PR 단위로 나누기 위한 초안입니다.

## Milestone 1. Baseline

- 저장소 scaffold
- backend health check
- frontend empty app
- Docker Compose MySQL
- CI

## Milestone 2. Auth And Ledger Foundation

- Kakao auth
- local/test-only developer login
- session API
- default personal ledger
- ledger list/switch
- protected frontend shell

## Milestone 3. Transaction Core

- category
- transaction create/update/query
- quick transaction
- calendar/ledger UI
- transaction edit UI

## Milestone 4. Budget And Dashboard

- ledger month settings
- member/category budget
- month close/reopen
- dashboard summary
- statistics

## Milestone 5. Collaboration

- group ledger
- direct invitation
- link invitation
- pending invitation
- invitation UI

## Milestone 6. Automation And Import

- recurring transactions
- due/generate flow
- transaction import preview
- backend Native Tesseract OCR와 이미지 전처리 integration

## Milestone 7. Product Readiness

- README update
- screenshots
- deployment docs
- full API examples
- seed data
- final verification
