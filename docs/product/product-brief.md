# Product Brief

## Product

우리로그는 개인과 공동체가 함께 사용할 수 있는 예산 운영 가계부입니다.

처음에는 기본 개인 장부로 바로 기록을 시작하고, 필요하면 여행, 생활비, 모임, 동거, 부부/연인 같은 목적별 공동 장부로 확장합니다.

## Problem

일반 가계부는 개인 지출 기록에는 적합하지만, 공동 지출을 다룰 때 다음 문제가 생깁니다.

- 누가 결제했는지와 공동 장부 예산이 함께 보이지 않습니다.
- 월 예산, 카테고리 예산, 실제 지출 흐름이 분리되어 있습니다.
- 반복 지출과 빠른 기록 흐름이 약하면 기록이 금방 밀립니다.
- 공동 장부 초대와 멤버별 사용 현황이 자연스럽게 연결되지 않습니다.

## Target Users

- 개인 장부를 쓰며 월 예산을 관리하려는 사용자
- 연인, 부부, 가족, 룸메이트처럼 생활비를 함께 관리하는 사용자
- 여행, 모임, 프로젝트처럼 기간이나 목적이 있는 공동 비용을 관리하는 사용자

## Product Direction

- 기록은 빠르게, 검토는 한눈에 보이게 만듭니다.
- 개인 장부와 공동 장부를 같은 사용 흐름 안에서 다룹니다.
- 월 단위 예산 운영을 중심에 두고, 거래/통계/마감을 연결합니다.
- 공동 장부에서는 결제자, 멤버, 초대, 사용 현황을 명확히 드러냅니다.

## Success Criteria

- 신규 사용자가 로그인 후 별도 설정 없이 기본 개인 장부에서 거래를 기록할 수 있습니다.
- 사용자가 개인/공동 장부를 전환해도 현재 장부 맥락을 잃지 않습니다.
- 사용자가 월 예산 대비 현재 지출과 카테고리별 흐름을 확인할 수 있습니다.
- 사용자가 공동 장부에 다른 사용자를 초대하고 함께 거래를 관리할 수 있습니다.
- 주요 기능은 API 계약, 테스트, 문서와 함께 유지됩니다.
